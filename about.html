Project Description

The Emoji Catalog Explorer is a two-page web application that dynamically displays a catalog of emojis using data fetched from a JSON file hosted on GitHub.

Technologies Used:
  • HTML5
  • CSS3
  • Bootstrap 5
  • JavaScript (ES6+)
  • Fetch API
  • Async/Await
Features Implemented:
  • Search Functionality
  • Category Filtering
  • Sorting (A→Z / Z→A)
  • Loading Animation
  • Responsive Design
  • Bootstrap Cards
